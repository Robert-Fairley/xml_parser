### General

This repo contains an XML parser with a specific function written in several different programming and scripting languages so that
they're readily workable for whatever environment required. They're written either to a similar function or so that they're 
easily modifiable.

### Files and Languages

 - **Compiled Languages**
    - C : xml_parser_c.c
    - D : xml_parser_d.d
 - **Interpreted Languages**
    - Python : xml_parser_py.py
